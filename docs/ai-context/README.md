# AI Context Documentation

This directory contains documentation files intended to provide context for AI coding assistants (Claude, Copilot, etc.) working on the bitKlavier2 codebase.

## Files

| File | Purpose |
|---|---|
| `CLAUDE.md` | Main project guide: build system, architecture, conventions, dependencies, CI/CD |
| `RESONANCE_THREADING.md` | Design document for `ResonanceBKSynthesiser` multi-threaded voice rendering |

## Usage

When starting a session with an AI assistant, provide the relevant file(s) as context. `CLAUDE.md` covers the full project; topic-specific files (like `RESONANCE_THREADING.md`) provide deeper detail for focused work.
